# 柒幻 扬琴（Dulcimer）

一个用于网页表单快速填充的 Chrome 扩展。  
你可以把常用文本（手机号、邮箱、地址、备注等）保存为条目，在网页输入框中一键填充。

## 功能特性

- 一键打开/关闭侧边栏
- 点击条目自动填充当前输入框
- 条目增删改查
- 条目拖拽排序（在“全部分组”视图下）
- 分组管理（创建、重命名、删除）
- 一条数据可属于多个分组
- 按分组筛选查看条目
- 导入/导出 JSON
- 侧边栏支持左右停靠切换
- 侧边栏宽度可鼠标拖拽调整
- 本地数据备份与自动恢复（降低数据丢失风险）

---

## 安装方式（开发者模式）

1. 打开 Chrome，进入 `chrome://extensions/`
2. 右上角开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本项目根目录（包含 `manifest.json` 的目录）
5. 安装完成后，工具栏会出现扩展图标

## 使用说明

### 1）打开侧边栏

- 在任意普通网页点击扩展图标即可显示/隐藏侧边栏  
- `chrome://`、`edge://` 等浏览器内部页面不支持注入

### 2）添加并填充数据

1. 在“添加新数据”中填写名称和内容
2. 可选择所属分组（支持多选）
3. 点击“添加数据”
4. 在网页上先聚焦一个输入框
5. 点击侧边栏中的数据条目即可自动填充

### 3）编辑 / 删除 / 排序

- 每条数据右侧提供编辑与删除按钮
- 在“全部分组”下可拖拽条目调整顺序
- 拖拽结束后自动保存顺序

### 4）分组管理

- 创建分组：在“分组管理”输入名称并点击“创建”
- 重命名分组：点击分组标签上的 `✎`
- 删除分组：点击分组标签上的 `×`
  - 删除分组不会删除条目本身，只会移除条目与该组的关联

### 5）侧边栏布局

- 左右切换：点击“导入”旁边的切换按钮（图标）
- 调整宽度：拖拽侧边栏边缘
- 布局状态会自动持久化（宽度、左右停靠位置）

### 6）导入/导出

- 导出：点击“导出”下载当前数据
- 导入：点击“导入”选择 JSON 文件
- 兼容两种导入格式：
  - 旧格式：数组（仅 `dataItems`）
  - 新格式：对象（含 `groups` + `dataItems`）

## 数据存储说明

扩展使用 `chrome.storage.local` 保存数据，核心字段包括：

- `dataItems`：主条目数据
- `groups`：分组数据
- `dataItemsBackup`：条目备份
- `groupsBackup`：分组备份
- `sidebarWidth`：侧边栏宽度
- `sidebarSide`：侧边栏停靠方向（left/right）

首次安装时仅在无历史数据情况下初始化，不会在更新时覆盖已有数据。

## 项目结构

- `manifest.json`：扩展配置（MV3）
- `background.js`：扩展图标点击、安装初始化
- `content.js`：注入侧边栏 iframe、处理页面填充、布局控制
- `sidebar.html`：侧边栏 UI 结构
- `sidebar.js`：侧边栏业务逻辑（条目/分组/导入导出）
- `styles.css`：样式定义
- `icons/`：扩展图标资源

## 常见问题

### Q1：为什么点击图标没反应？

- 当前页面可能是浏览器内部页（如 `chrome://`），该类页面不支持注入扩展脚本。

### Q2：数据会不会丢失？

- 数据保存在 `chrome.storage.local`，并有备份键自动兜底恢复。
- 但如果卸载扩展，浏览器通常会清除此扩展本地存储。

### Q3：拖拽排序为什么在某些视图下不可用？

- 仅“全部分组”视图支持排序，分组筛选视图默认不进行全局排序。
